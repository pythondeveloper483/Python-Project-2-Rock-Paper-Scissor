## 🎮 About the Game
A simple **Python-based Rock-Paper-Scissors game** where you play against the computer.  
Choices are mapped to numbers for easy comparison:
- `1` → Rock  
- `-1` → Paper  
- `0` → Scissor  

The computer randomly selects its move, and the winner is decided based on classic rules.

---

## 🚀 Features
- 🎲 Random computer choice using `random.choice`
- 🖥️ User input with shorthand keys (`r`, `p`, `s`)
- ✅ Clear winner/loser/draw logic
- 🔄 Easy to expand with more features
